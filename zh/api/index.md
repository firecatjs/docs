# API

## FreCat

```ts
const app = new FireCat({
  bodyParserConfig?: BodyParserOptions, // 传入的 BodyParserOptions
  koaConfig?: FireCatKoaFace, // 传入的 KoaJsOptions
})
```

### 详细介绍

| 参数名            | 类型                | 说明                       |
| ----------------- | ------------------- | -------------------------- |
| `bodyParserConfig`| `BodyParserOptions` | 传入的 BodyParserOptions   |
| `koaConfig`       | `FireCatKoaFace`    | 传入的 KoaJsOptions        |
